implements 关键字在 class 声明中使用，以指示所声明的类提供了在 implements 关键字后面的名称所指定的接口中所声明的所有方法的实现。
示例-实现一个接口
public class Truck implements IVehicle
{
}
示例-实现多个接口
public class Truck implements IVehicle1,IVehicle2,IVehicle3
{
}
 注释
在上例中，Truck 类必须提供在 IVehicle 接口中所声明的所有方法的实现。 否则，Truck 类将是独立的；它可以声明其他方法和变量，并扩展另一个类。
一个类可以实现多个接口。
